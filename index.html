Tags de Título

<h1>

Representa o título principal da página.

<h2>

Utilizada para subtítulos importantes.

<h3>

Utilizada para seções dentro de um subtítulo.

<h4>

Representa um nível menor de título.

<h5>

Utilizada para títulos com menor destaque.

<h6>

A tag <p> é utilizada para criar parágrafos de texto.

A tag <mark> serve para destacar uma parte do texto.

A tag <strong> destaca o texto com forte importância.
